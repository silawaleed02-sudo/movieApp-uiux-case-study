# 🎬 Movie Streaming App – UI/UX Case Study

## 📖 Overview
This project is a complete UI/UX case study for a movie streaming application inspired by platforms like Netflix. The goal was to create a modern, user-friendly experience that makes discovering and watching movies simple and enjoyable.

## ✨ Project Includes
- User Research
- User Persona
- User Journey Map
- User Flow
- Wireframes
- High-Fidelity UI Design
- Style Guide

## 🛠️ Tools
- Figma
